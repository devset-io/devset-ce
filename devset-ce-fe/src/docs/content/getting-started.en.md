# Getting Started

If you are starting from scratch, follow this path step by step.

## Full process overview

![Getting Started - process overview](/docs/start-workflow-journey.svg)

## What to do first

First, add `Schema Repo` and upload your JSON schema.
This is the starting point for the entire flow.

## Step by step

1. In `Flow Builder`, build the flow you need.
2. In `Playground`, verify the expected results and confirm that the logic is correct.
3. In `Brokers`, define a broker (host, login, and so on).
4. In `Workflow Runs`, choose a workflow and broker, then start a run.

## Where to do what

| Section | What it is used for |
| --- | --- |
| Schema Repo | Add and maintain JSON schemas |
| Flow Builder | Build and edit flows |
| Playground | Test the result before a run |
| Brokers | Configure the broker connection |
| Workflow Runs | Start and monitor runs |
