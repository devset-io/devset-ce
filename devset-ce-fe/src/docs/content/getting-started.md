# Start

Jesli zaczynasz od zera, przejdz ta sciezke krok po kroku.

## Schemat calego procesu

![Start - schema procesu](/docs/start-workflow-journey.svg)

## Co najpierw

Najpierw dodaj `Schema Repo` i wrzuc swoj schemat JSON.
To jest punkt startowy dla calego flow.

## Krok po kroku

1. W `Flow Builder` budujesz interesujacy Cie flow.
2. W `Playground` sprawdzasz, jakie beda wyniki i czy logika jest OK.
3. W `Brokers` definiujesz brokera (host, login, itp.).
4. W `Workflow Runs` wybierasz workflow + brokera i uruchamiasz run.

## Gdzie co robisz

| Sekcja | Do czego sluzy |
| --- | --- |
| Schema Repo | Dodanie i utrzymanie schem JSON |
| Flow Builder | Budowa i edycja flow |
| Playground | Testowanie wyniku przed runem |
| Brokers | Konfiguracja polaczenia do brokera |
| Workflow Runs | Odpalanie i obserwacja uruchomien |
