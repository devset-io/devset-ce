# Flow Builder Management

## 1. Flow Builder Management

This is the workflow management screen. Here you can see the catalog of all saved flows and decide what to do with them next.

### What you can see on this screen

- **Section header** `Flow Builder Management` with a short description.
- **Workflow count** in the catalog (for example `9 workflows in catalog`).
- **Primary buttons** in the top-right corner:
  - `Import from JSON`
  - `Create New Workflow`
- **Workflow table** with columns:
  - `Workflow ID`
  - `Label`
  - `Actions`
- **Actions for each row**:
  - `Open`
  - `Clone`
  - `Delete`

### What you can do here

1. Create a new workflow with `Create New Workflow`.
2. Upload an existing workflow from JSON with `Import from JSON`.
3. Open an existing workflow for editing with `Open`.
4. Clone a workflow and prepare a new version with `Clone`.
5. Delete a workflow that is no longer needed with `Delete`.

### How to use it in practice

- If you are starting from scratch: click `Create New Workflow`.
- If you already have a payload: click `Import from JSON`.
- If you want to update an existing process: click `Open`.
- If you want to make a variant without touching the original: click `Clone`.
