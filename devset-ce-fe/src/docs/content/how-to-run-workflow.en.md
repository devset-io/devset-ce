# How to run a workflow

## Step by step

1. Go to **Flow Builder** and save the workflow.
2. Open **Workflow Runs** and click `New run`.
3. Fill in the required event parameters.
4. Observe the execution status and result.

## Common issues

- Without a schema in `Schema Repo`, you cannot add a stage.
- An unsaved draft in `Flow Builder` prevents quick switching between views.
