# Jak uruchomić workflow

## Krok po kroku

1. Wejdź do **Flow Builder** i zapisz workflow.
2. Otwórz **Workflow Runs** i kliknij `New run`.
3. Uzupełnij wymagane parametry eventu.
4. Obserwuj status i wynik wykonania.

## Typowe problemy

- Brak schematu w `Schema Repo` powoduje brak możliwości dodania stage.
- Niezapisany draft w `Flow Builder` blokuje szybkie przejście między widokami.
