## [Unreleased]

### Added
- Initial source-available release of Devset CE
- Backend: Spring Boot workflow engine with Kafka, RabbitMQ, and MongoDB connectors
- Backend: Execution plan engine with DSL-driven workflow steps
- Backend: Protobuf schema support and wire-format encoding
- Backend: SQLite persistence with Ehcache caching
- Frontend: React-based Flow Builder with visual canvas editor
- Frontend: Message Dispatch for sending messages to brokers
- Frontend: Kafka Live view for real-time topic streaming
- Frontend: Schema Repository management
- Frontend: Workflow Runs monitoring with SSE event feed
- Frontend: Function Studio for composing workflow steps
- CI/CD: GitHub Actions for build, test, and release automation
- Docker support with multi-stage build
